# Mémo Git

## Nouveau projet
### Crée un répertoire
`git init`
### Voir le working directory
`git status`
### Ajouter des fichiers
`git add`
### Commit
`git commit -m "La raison du commit"`
### Conexion avec le Repertory
`git remote add origin URL-DU-GITHUB`
### Changer de branch
`git branch -M main`
### Envoyer sur le Repetory
`git push -u origin main`
